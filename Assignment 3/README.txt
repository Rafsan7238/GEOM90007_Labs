Please follow the following instructions to start the interface:

1. Open the Server.R file in R Studio or any other suitable IDE. 
2. Make sure the libraries marked by the command 'library(...)' are installed in your system.
3. Click on "Run App". This should run the scripts and open the dashboard in a browser. 
4. Copy the URL of the opened dashboard window. 
5. Open the Melbourne Explorer.twbx file. 
6. Go to Hotels DB dashboard page. Click on the tile below the page banner, then click on the down arrow "more options button and select 'Edit URL', and paste the URL copied from R dashboard window. This should open the R dashboard on Tableau. 
7. Go to the "Melbourne Explorer Storyboard" page on Tableau, and you may now interact with it to see the different features of the interface.
